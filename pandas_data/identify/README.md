# 百度图片识别API + 浏览器交互`pywebio`

## 项目功能
* 使用百度图片识别API
* 使用`pywebio`将浏览器作为图片输入及识别结果的展示终端
* 使用`pyinstaller`将`python`程序转换为`exe`程序
* 制作`exe`程序的`ico`图标

>打包exe命令：

`pyinstaller -F -i ic.ico baidu.py`
